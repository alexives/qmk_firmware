# Ives Dox Keymap

This keymap is designed in mind for people who are primarily OS X users. Like the default planck map, the four base layers represent the four main keyboard layout types, Qwerty, Workman, Colemak, and Dvorak. Unlike the default planck map, the number row has been moved to the home row in the raise layer, and the associated punctuation has been moved to the lower layer. This mapping is not 1-1 to the number row shift, and some of the programmer focused punctuation has been arranged to be a bit more forward focused.

Unlike the planck, obviously this keymap still has it's number row. So the number row was kept as is. Also, there are a lot more keys to use, so it uses them. Including dedicated parenthesis, dedicated square brackets, and a variety of media and convenience keys.

This graphic represents the layers on the Qwerty base, however when powered, you can specify a different base layer (which is not persisted).

[![the layout](https://raw.githubusercontent.com/alexives/qmk_firmware/ives_layouts/keyboards/ergodox_ez/keymaps/ives/ives-ergodox.png)](http://www.keyboard-layout-editor.com/#/gists/7c5d87cc4c3506d8b71e)
